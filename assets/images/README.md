# Fotos del proyecto

Personaliza fotos solo aquí:

- `assets/images/images.js`

Pasos simples:

1. Copia tus imágenes en `assets/images/index/` y `assets/images/menu/`.
2. Actualiza las rutas en `assets/images/images.js`.
3. Mantén extensiones correctas (`.jpg`, `.png`, `.webp`).

Si una ruta falla, la web mostrará `assets/images/fallback.svg` automáticamente.
